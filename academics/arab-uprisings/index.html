    Project Summary

    I looked at the relationship between social media and the revolutions of the Arab uprisings in Tunisia, Egypt, and Libya for my Master’s thesis in Global Technology and Development (GTD). The project consisted of the following work:

    • Designed and implemented a structured research plan
    • Built a website to host static blog posts on Github using Jekyll
      • Installed citation management plugin using BibTeX
      • Built basic site tests using Travis CI
    • Composed and delivered 15 posts on the topic
    • Shared the blog posts with peers via the GTD Facebook group and on Twitter
    • Performed a formative evaluation of the project data (Twitter, Facebook, and Google Analytics)
